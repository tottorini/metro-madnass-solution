# BankAccountManager: Stakeholder Requirements

Abstract: This document contains the requirments for a bank account manager system.

1. Accounts
    + The users should be able to open new accounts.
    + The users should be able to close existing accounts.
    + Each account must have a unique identifier.
    + Each account must have three
2. Actions
    + Users should be able to withdraw funds.
    + Users should be able to deposit funds.
    + Users should be able to transfer funds to different accounts.
    + The system shall ensure that transfers are only possible if the source account has sufficient funds.
    + The user should be able to see updated funds after transfers.
    + The system must handle invalid inputs.
3. Performance
    + The system should support concurrent access by multiple threads.
    + The system must ensure data integrity
